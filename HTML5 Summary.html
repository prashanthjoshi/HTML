HTML5 Summary and Best Practices

  • DOCTYPE specifies the html version by default html5
  • Meta tag is not visible on browser but machine readable and provides info related page, set viewport and author details.
  • Tilte tag defines the title to be displayed on browser tab
  • Style tag defines style information for the webpage
  • Link tag defines a link to other documents
  • Script tag defines JavaScript and other script related code or file

Attributes

  • Html tags can have various additional properties called attributes.
  • Some attributes like name and value are common to most of the elements and can be classified as global attributes.
  • Attributes are set only in start tag of elements.
  • Best practice is to use lower case.
  • use double quotes "" on attribute values as they might not work as intended if contained any special chars.
