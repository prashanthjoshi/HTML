Attributes

  • Html tags can have various additional properties called attributes.
  • Some attributes like name and value are common to most of the elements and can be classified as global attributes.
  • Attributes are set only in start tag of elements.
  • Best practice is to use lower case.
  • use double quotes "" on attribute values as they might not work as intended if contained any special chars.
